# Запуск проекта

## 1. Установи Node.js
https://nodejs.org — скачай LTS версию

## 2. Установи Expo CLI
```
npm install -g expo-cli
```

## 3. Установи зависимости проекта
```
cd "Character Chat/CharacterChatRN"
npm install
```

## 4. Запусти
```
n
```

## 5. Открой на телефоне
- Скачай **Expo Go** (App Store / Google Play)
- Отсканируй QR-код из терминала

## Или на эмуляторе
- iOS симулятор: нажми `i` в терминале (нужен Xcode, только macOS)
- Android эмулятор: нажми `a` в терминале (нужен Android Studio)

---

**Если ошибка при npm install** — попробуй:
```
npm install --legacy-peer-deps
```
